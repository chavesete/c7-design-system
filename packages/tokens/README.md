# @chavesete/tokens

Tokens do Design System da Chave7 para consumo em web e mobile.

## Instalação

```bash
npm i @chavesete/tokens
```

## Web (CSS vars)

Importe no CSS global do app:

```css
@import "@chavesete/tokens/styles.css";
```

## JS/TS

```ts
import { c7CssVars, c7NativeColors } from "@chavesete/tokens";
```
