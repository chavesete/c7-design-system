# Tipografia

Hierarquia, pesos e font-family padrão.

## Font family

O Storybook e os exemplos web usam:

- Metropolis (fallback: Inter, system-ui)

## Exemplos

<div className="rounded-lg border border-c7-border bg-c7-card p-6">

## Font sizes do design system

Além da escala padrão do Tailwind, o preset expõe os seguintes tamanhos:

| Classe | Tamanho | Line height |
|---|---|---|
| `text-c7-small` | 10px | 1.4 |
| `text-c7-subtitle` | 12px | 1.4 |
| `text-c7-body` | 14px | 1.5 |
| `text-c7-body-lg` | 16px | 1.5 |
| `text-c7-title` | 18px / bold | 1.3 |

## Boas práticas

- Priorize contraste: `text-c7-foreground` em `bg-c7-background`/`bg-c7-card`
- Use `text-c7-muted-foreground` para textos secundários
- Evite variações "soltas" de tamanho: prefira a escala do Tailwind ou as classes `text-c7-*` acima
