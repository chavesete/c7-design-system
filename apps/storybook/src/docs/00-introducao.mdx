## Estrutura do monorepo

- `packages/tokens`: tokens e variáveis CSS do tema (web) + tokens JS/TS (mobile)
- `packages/tailwind-preset`: presets Tailwind (web e native)
- `packages/ui`: componentes web (React + Tailwind)
- `packages/ui-native`: componentes mobile (React Native + NativeWind)
- `apps/storybook`: documentação (este Storybook)

## Temas (light/dark)

No web, o tema é definido via CSS variables (HSL) e o modo escuro é ativado aplicando a classe `dark`
no elemento raiz (ex.: `html` ou `body`).
