# Espaçamento e raio

Padrões de layout usando Tailwind + tokens de raio.

## Espaçamento

Para espaçamentos, use a escala do Tailwind (`p-*`, `m-*`, `gap-*`, `space-y-*`).

Recomendações práticas:

- Layouts: `gap-4` / `gap-6`
- Cards: `p-4` (densidade padrão), `p-6` (conteúdo rico)
- Listas: `space-y-2` / `space-y-3`

<div className="grid grid-cols-1 gap-4 lg:grid-cols-3">

    <div className="mt-3 space-y-2">

## Raio (border-radius)

O DS expõe tokens de raio via CSS variables (web):

- `--radius`
- `--radius-card`
- `--radius-sheet`

<div className="mt-4 grid grid-cols-1 gap-4 sm:grid-cols-3">
  <div className="rounded border border-border bg-card p-4">
    <div className="text-sm font-semibold">radius</div>
    <div className="mt-3 h-14 border border-border" style={{ borderRadius: "var(--radius)" }} />
    <div className="mt-2 text-xs text-muted-foreground">var(--radius)</div>
  </div>

  <div className="rounded border border-border bg-card p-4">
    <div className="text-sm font-semibold">radius-card</div>
    <div className="mt-3 h-14 border border-border" style={{ borderRadius: "var(--radius-card)" }} />
    <div className="mt-2 text-xs text-muted-foreground">var(--radius-card)</div>
  </div>

  <div className="rounded border border-border bg-card p-4">
    <div className="text-sm font-semibold">radius-sheet</div>
    <div className="mt-3 h-14 border border-border" style={{ borderRadius: "var(--radius-sheet)" }} />
    <div className="mt-2 text-xs text-muted-foreground">var(--radius-sheet)</div>
  </div>
</div>
